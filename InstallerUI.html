Next Steps

On Google Docs...

1) Log into Google Drive and find your new GoogPress folder. This is where everything lives. Inside there should be a file called 'GoogPress' and a folder called 'Posts'. Open up 'GoogPress' (it might ask you if you want to use a 3rd party app - choose 'Google Apps Script').



2) It should open in something vaguely remniscient of a text editor. Find the menu bar and navigate to 'Publish > Deploy as web app'.



3) Make sure the app is set to execute as 'Me' and that 'Anyone, even anonymous' can access the app (this is so that we can send it requests).



4) Hit deploy, grant it the necessary permissions and then copy the web app URL and put it somewhere safe.



5) You can now add posts to your GoogPress blog by dragging them into the posts folder on Drive (see the first picture - it's that).
